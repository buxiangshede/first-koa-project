first-koa-project

This mini app showcases common Koa patterns: custom middleware, routing, JSON parsing, and static files.

Available routes

  • / overview response
  • /health heartbeat
  • /features list of highlighted Koa concepts
  • /users list and POST new users (body: {"name":"","role":""})
  • /users/:id fetch a single user
  • /users/:id/preferences update arbitrary preferences
  • /echo?name=Koa inspect query parameters
  • /slow demonstrates async work inside middleware

Happy hacking!
